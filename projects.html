Amazon Remake
After another tutorial from SuperSimpleDev, I was able to learn how to use JavaScript with HTML and CSS to create an interatice website. This project is a remake of Amazon.com. On my website, you are currently able to add items to your cart and remove them, and the cart will update with the correct number of items. I learned how to use JavaScript to manipulate the DOM and create an interactive website. I also learned how to use the localStorage to store the items in the cart so that when you refresh the page, the items will still be in the cart. I still have more work to do on this project, but I am proud of what I have accomplished so far.

Youtube Remake
With the help of the youtuber SuperSimpleDev, I was able to learn HTML and CSS and create this project that recreates youtube.com. I learned many of the basic components of these two language. With HTML, I used divs to create the grid of the "for you" videos, and also to create the header and sidebar using absolute positioning. CSS was used to style the page and make it look like the real youtube.com. To me, the most important tools that I learned are the following: nested layouts, flexbox and grid, media queries, positioning, and the box model. Even though this project isn't interactive, I learned a lot about the basics of HTML and CSS and how to use them to create a website.
